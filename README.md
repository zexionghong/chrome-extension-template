# Chrome 代理扩展

基于 Vite + Vue3 + TypeScript 的 Chrome 代理扩展

## 功能特性

- 🚀 基于 Vite 构建,开发体验好,构建速度快
- 🎯 使用 TypeScript 开发,类型安全
- 🎨 集成 Vue3 框架,组件化开发
- 📦 代理管理功能:
  - popup - 快速切换代理配置的弹出面板
  - background - 代理规则匹配和转发后台脚本
- 🔧 完整的开发工具链:
  - ESLint + Prettier 代码规范
  - TypeScript 类型检查
  - 热更新开发
  - 自动导入 API
- 🔌 代理相关工具类:
  - Chrome代理API封装 - `chrome-proxy`
  - 代理配置存储 - `proxy-storage`
